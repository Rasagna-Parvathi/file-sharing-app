# 📁 File Sharing Web Application

## 📌 Project Description
This is a simple file sharing web application that allows users to upload files and generate a downloadable link. The application is built using Node.js and Express, with Multer used for handling file uploads.

## 🚀 Features
- Upload files easily
- Generate shareable download links
- Download files using link

## 🛠️ Tech Stack
- Frontend: HTML, JavaScript  
- Backend: Node.js, Express  
- Middleware: Multer  

## 📂 Project Structure
FileSharingApp/
│── backend/
│   ├── server.js
│   ├── package.json
│   ├── uploads/
│── index.html

## ⚙️ How to Run the Project
1. Clone the repository  
   git clone https://github.com/Rasagna-Parvathi/file-sharing-app.git  

2. Go to backend folder  
   cd backend  

3. Install dependencies  
   npm install  

4. Run server  
   node server.js  

5. Open index.html in your browser  

## 🎯 Output
- Upload a file  
- Get a download link  
- Click link to download the file
